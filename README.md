# Русские шашки онлайн

Веб-версия русских шашек для двух игроков. Один игрок создает комнату, второй подключается по коду или ссылке.

## Запуск

```bash
node server.js
```

После запуска откройте:

```text
http://127.0.0.1:3000
```

Если нужно принимать подключения извне на сервере или в контейнере:

```bash
HOST=0.0.0.0 PORT=3000 node server.js
```

## Что уже есть

- комнаты с коротким кодом;
- случайное распределение белых и черных при создании комнаты;
- партия начинается только после подключения второго игрока;
- третьи посетители становятся наблюдателями;
- обязательное взятие;
- серии взятий;
- дамки с дальним ходом и дальним взятием;
- превращение простой шашки в дамку;
- серверная проверка ходов.
